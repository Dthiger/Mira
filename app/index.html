Mira shortcuts

Tools

Brush
Paint with the active color. Press and drag. B
Lasso
Click to add control points; the outline is a smooth spline through them, the fill is hard-edged. L
Paint bucket
Flood-fill the connected region under the cursor. G
Erase all
Brush-shaped tool that erases anything touched. E
Erase selected
Brush-shaped tool that only erases pixels of the currently-selected color (magic eraser). Shift+E
Quick erase
Hold Ctrl while using the brush for a one-off erase that lasts only while the key is held.

Lasso details

Add point
Click on the canvas
Freehand trace
Click and drag — points are added along the path
Close shape
Click on the first point's blue ring, double-click anywhere, or press Enter
Cancel
Esc

Viewport

Pan
Middle-mouse drag (unbounded — push the corner past the viewport if you like)
Zoom
Mouse wheel — anchored at the cursor
Cycle preset zoom
Click the zoom percentage in the bottom-right (cycles 50% → 100% → 200%, snapping to the closest preset before advancing)
Fit to view
F or the fit button (bottom-right)
Change brush size
Right-mouse drag (horizontal), or Alt + middle-mouse drag. Brush-size popover only appears under brush/eraser tool buttons.

History

Undo
Ctrl + Z
Redo
Ctrl + Shift + Z or Ctrl + Y

Color

Pick color
Dropdown lists 001–025 sorted from pure green to pure red
Cycle color
Shift + mouse wheel (wrap-around)
Used-color highlight
Indices already painted somewhere in the document are tinted teal in the dropdown list, so you can see at a glance which colors are in use.

Reference image

Load
Image-icon button in the bottom status bar (labelled "Ref")
Opacity
Slider next to the load button. The reference is composited on top of the painted mask, so opacity controls how strongly it shows through your work.

Status bar (bottom)

POS
Integer document-pixel coordinates under the cursor.
ID
Swatch + label + name of the palette index at the cursor pixel (or "background"). Live as you hover.
USED N
Number of distinct palette indices currently present in the document. Hover the cluster to see the full list with swatches, names, and pixel counts.

Save / Load / Export

Save state
Teal floppy button next to the Mira logo (top-left). Writes the current mask as a plain PNG (palette colors + black background). Open it in any image viewer to preview; reopen it in Mira to resume editing.
Load state
Folder-open button next to the save button. Pick a previously-saved Mira state PNG. Snapshots history first so Ctrl+Z still undoes the load.
EXPORT
Text button at the top-right. Produces the 16-bit RGBA distance-fill PNG plus a JSON sidecar. The PNG encodes: R/G = palette color of the nearest connected region, B = white for background / black for painted (inverted for shader subtraction), A = smoothstep distance gradient with the shape edge pinned at midpoint 32768. Sidecar JSON lists every palette ID present, its label, name, RGB, and pixel count.

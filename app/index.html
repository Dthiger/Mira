Mira shortcuts

Tools

Brush
Paint with the active color. Press and drag. B
Lasso
Click to add control points; the outline is a smooth spline through them, the fill is hard-edged. L
Paint bucket
Flood-fill the connected region under the cursor. G
Erase all
All tools subtract anything they touch. E
Erase selected
All tools subtract only pixels of the currently-selected color (magic eraser). Shift+E
Quick erase
Hold Ctrl while drawing for a one-off erase-all that lasts only while the key is held.

Close shape

Cancel

Reference opacity
Slider next to the loader

Export

Distance-fill PNG
16-bit RGBA: R/G = palette color of the nearest connected region, B = white for background / black for painted, A = smoothstep distance gradient (midpoint 32768 is the shape edge). A JSON sidecar with the same base filename ships alongside, listing which palette IDs are present, their pixel counts, and the export timestamp.
